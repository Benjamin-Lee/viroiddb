ViroidDB

Viroids and their (potential) relatives are among the most unique and exciting of all biological entities. However, I couldn't find a good, up-to-date database containing all of them in one place so I made my own.

This database has sequences representing known viroid and viroid-like RNA sequences. Each sequence is unique, though variants of the same RNA species are included. Currently, sequences are not guaranteed to be rotationally unique, meaning that AAAATTTT is treated as distinct from TTAAAATT.

How to get the data

You can download the most recent version of the entire database as a FASTA file here.

Alternatively, download the database easily using your command-line tool of choice: 

$ curl viroids.org/db/all.fasta > viroiddb.fasta

How I curate the database

All data is publicly available from the NCBI. I have included what I believe to be a class of potentially related RNA agents that merit study as a group. The key inclusion criteria are:

  • Circularity at some point in the replication cycle
  • Extensive RNA folding
  • The presence of ribozymes or a central conserved region

To collect the sequences for the database I followed this protocol:

  1. For viroids and deltaviruses, I used the NCBI Virus portal to download complete sequences within the taxonomy IDs 185751, 185752, 265963 and 39759 for the Pospiviroidae, Avsunviroidae, unclassified viroids, and Deltavirus taxa respectively.
  2. For retrozymes, which are not taxonomically classified, I downloaded all sequences within the NCBI Nucleotide database matching the search term "ribozyme".
  3. Similarly, for satellite RNAs, I downloaded all sequences within the NCBI Nucleotide database matching each of the small circular satellites in Viroids and Satellites (2017).
  4. Deduplication and formatting is performed by SeqKit. Comment lines are standardized as >accession genbank_title.

All the downloading (and website updating) is currently done manually. Until NCBI Virus has an API, I will repeat this process occasionally and update this site. Should new viroid-like RNAs classes be discovered, I will add them as a new subset. If you feel an update is required, please contact me or perform the update yourself and submit it as a pull request to the repository.
